Для веб-программирования (базы) хорошо:

1) уметь настраивать веб-сервера: связку nginx+apache2 (httpd)
nginx - как front-end, раздающий статику
apache2 с модулем php - как bach-end, выполняющий статику.
Задача настроить так, чтобы статические файлы отдавал быстрый nginx, а динамика проксировалась на back-end и выполнялась php.

vrajendra.ru сейчас именно так работает. Это позволяет нам очень сильно экономить, беря менее мощный сервер (5usd/месяц против 100usd/месяц).

2) Базы данных SQL.
Серверная MySQL, локальная - SQLite. 
Важная часть в БД - теория проектирования баз данных ("3-ая нормальная форма" и т.д.)

3) Скриптовые языки: php, python, java + html верстка, javascript для сайтов, основы понимания разметки css.

4) Пользоваться ssh для удаленного администрирования серверов из коммандной строки. (рекомендую Midnight Commander сразу поставить).
